Call, Apply & Bind

Añadir contexto a funciones

Estos tres métodos se usan para controlar la llamada de la función. call() y apply() aparecieron en ECMAScript 3 y bind() en ESCMAScript 5.

Se puede usar call() / apply() para llamar a una función inmediatamente. bind() devuelve una función que, una vez ejectutada, tiene el uso correcto de this.

En javascript, las funciones

Consola Javascript

/advanced/7_call_apply_bind/index.js

console.log("Call, Apply & Bind");

// Call
console.log("Call");

var myObj = {
    num: 2
}

var addNumbers = function(num1, num2) {
    return this.num + num1 + num2
}

console.log(addNumbers());

// We can use call to invoke a function attaching the context with some object (and the rest of the parameters)
console.log(addNumbers.call(myObj, 1, 2));


// Apply
// Same concept as call but passing the parameters as an array
console.log("Apply");

let params = [1, 2]

console.log(addNumbers.apply(myObj, params));

// Bind
// Returns a new function with the binding between the object and the function
// Is used for example to have the context of the object in a callback

console.log("Bind");
var otherObj = {
    num1: 5,
    num2: 4
}

var otherFunc = function() {
    return this.num1 + this.num2;
}

var returnedFunc = otherFunc.bind(otherObj);

console.log(returnedFunc());
